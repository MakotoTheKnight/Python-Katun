What is K'atun?

K'atun is intended to be an open-source, platform-agnostic music organizer. Our intention is to allow someone with a large collection sort through all of the duplicates, as well as add correct tagging, with support from MusicBrainz. Currently, this software is in early alpha, so features you see here are incomplete. Please visit the Google Code Website to follow this developing software.
